Neurona is the code for creating a Bluetooth MIDI musical interface. 

It is designed to run on the ESP32 board. 

The schematic and construction instructions can be found at 

https://danisantos.es/research/   

It features four potentiometers and six buttons. 
It is ideal for controlling effects in DAWs such as Ableton, Reaper, Logic, FL Studio, etc. 
Lightweight, compact, extremely useful, and relatively easy to build.
